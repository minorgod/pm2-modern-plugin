## Development Quick Start

- Git clone:
    - `git clone https://github.com/Shelob9/wordpress-plugin`
- Composer install
    - `composer install`
- Yarn install.
    - `yarn`
    - [Use Yarn 1.x, not npm](https://dev.to/shelob9/why-i-use-yarn-not-npm-dkk)
- Build JS/CSS
    - `yarn build`
- Ensure unit tests pass
    - `composer test`
- Start WordPress
    - `npx wp-env start`
